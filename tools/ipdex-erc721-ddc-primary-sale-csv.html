IPDEX ERC-721 (DDC) primary sale — NFT CSV generator

Generates a UTF-8 CSV compatible with Admin POST /api/v1/create/primary/sale NFT upload (parseNfts): one row per minted ERC-721 unit, supply always 1, tokenId as plain decimal (no leading zeros). Open this file locally in a browser (double-click or file://); nothing is uploaded.

Backend checks: sum(supply) + c_circulating_supply ≤ c_total_supply on the collection.

Each next row increments by 1. Must satisfy IPDEX isUint256String.

Placeholders: {{tokenId}}, {{index}} (0-based row index).

If empty, rows use {"name","description","image","attributes":[]} aligned with typical OpenSea-style JSON.
If you provide JSON, only string values are scanned for placeholders.
